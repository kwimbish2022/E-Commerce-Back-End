# E-Commerce-Back-End

## Description

Make a back-end for ecommerce. Use Express.js API, dotenv, and Sequelize to work with a MySql database.

## Installation
`npm init`

`npm install mysql2`

`npm install sequelize`

`npm install dotenv`
  
## Usage
Run command at project root:

`mysql -u root -p`

Enter when promted

`source db/schema.sql`

`quit`

`npm run seed`
  
`npm start`
